# library/mcps/

MCP (Model Context Protocol) catalog, setup recipes, and reference docs.

- `catalog.md` — index of all known MCPs with status and purpose
- `recipes/` — shell snippets and config templates for installing MCPs
- `opencode-mcp-helper.sh` — helper for configuring MCPs in opencode

When adding a new MCP, add an entry to `catalog.md` and a recipe to `recipes/`.
