# go-blueprint

> Go-blueprint allows users to spin up a quick Go project using a popular framework

## What's Inside
- CLI tool to generate Go project structure
- Framework selection: Chi, Gin, Fiber, HttpRouter, Gorilla/mux, Echo
- Database drivers: MySQL, Postgres, SQLite, Mongo, Redis, ScyllaDB
- Advanced features (via --advanced flag):
  - HTMX + Templ support
  - GitHub Actions CI/CD
  - Websocket endpoint setup
  - Tailwind CSS integration
  - Docker configuration
  - React frontend (TypeScript)
- Blueprint UI web app for previewing structure
- Install via Go, npm, or Homebrew

## Stack
- Go, Cobra CLI
- Multiple framework support
- Multiple database drivers
- GitHub Actions, Docker
- React, TypeScript (optional)
