# Open SaaS

> A 100% free modern JS SaaS boilerplate (React, NodeJS, Prisma). Full-featured: Auth (email, google, github, slack, MS), Email sending, Background jobs, Landing page, Payments (Stripe, Polar.sh), Shadcn UI, S3 file upload. AI-ready with tailored AGENTS.md, skills, and Claude Code plugin. One cmd deploy. Powered by Wasp full-stack framework.

## What's Inside
- Wasp full-stack framework (React + Node.js + Prisma)
- Astro Starlight for documentation and blog
- Authentication: email, Google, GitHub, Slack, Microsoft
- Payments: Stripe, Polar.sh, Lemon Squeezy
- Background jobs with pg-boss
- Email sending with SendGrid, MailGun, or SMTP
- AWS S3 file uploads
- Shadcn UI with TailAdmin dashboard
- OpenAI API integration with function calling
- Analytics: Plausible or Google Analytics
- End-to-end tests with Playwright
- One-command deployment to Railway or Fly.io

## Stack
