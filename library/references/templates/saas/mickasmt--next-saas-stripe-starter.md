# Next SaaS Stripe Starter

> Open-source SaaS Starter with User Roles & Admin Panel. Built using Next.js 14, Prisma, Neon, Auth.js v5, Resend, React Email, Shadcn/ui, Stripe, Server Actions.

## What's Inside
- Next.js 14 with App Router
- Authentication with Auth.js v5 (GitHub, Google OAuth)
- Database with Prisma ORM and Neon serverless Postgres
- Stripe payment integration with webhooks
- Email with Resend and React Email
- Shadcn/ui components with Tailwind CSS
- Server Actions for mutations
- User roles and admin panel
- Contentlayer for MDX content
- Framer Motion for animations

## Stack
- TypeScript, MDX
- Next.js 14
- Prisma ORM
- Neon (serverless Postgres)
- Auth.js v5
- Stripe
- Resend, React Email
- Shadcn/ui, Tailwind CSS
- Framer Motion, Lucide icons
