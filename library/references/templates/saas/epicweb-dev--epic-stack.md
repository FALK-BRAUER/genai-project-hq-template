# Epic Stack

> This is a Full Stack app starter with the foundational things setup and configured for you to hit the ground running on your next EPIC idea.

## What's Inside
- Remix web framework with React Router
- Fly.io deployment with Docker
- Multi-region SQLite database with LiteFS
- Email/password authentication with cookie-based sessions
- Database ORM with Prisma
- End-to-end testing with Playwright
- Unit testing with Vitest and Testing Library
- Styling with Tailwind CSS and Radix UI components
- GitHub Actions CI/CD with staging and production environments

## Stack
- TypeScript, JavaScript, CSS
- Remix (React framework)
- Prisma ORM
- SQLite with LiteFS
- Tailwind CSS, Radix UI
- Playwright, Vitest
- Fly.io
