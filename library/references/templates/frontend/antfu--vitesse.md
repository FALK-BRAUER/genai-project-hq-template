# vitesse

> Opinionated Vite + Vue Starter Template

## What's Inside
- Vue 3 + Vite + pnpm + esbuild for fast development
- File-based routing with `unplugin-vue-router`
- Components auto-importing
- Pinia for state management
- UnoCSS for on-demand atomic CSS
- I18n ready with Vue I18n
- Markdown support via `unplugin-vue-markdown`
- PWA support with `vite-plugin-pwa`
- Unit testing with Vitest, E2E with Cypress
- Netlify deploy ready

## Stack
- Vue 3, Vite, TypeScript
- UnoCSS, Iconify
- Pinia, Vue Router
- Vitest, Cypress
