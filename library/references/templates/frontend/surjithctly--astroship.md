# astroship

> Astroship is a starter template for startups, marketing websites, landing pages & blog

## What's Inside
- Astro 3 + Tailwind CSS
- Content Collections API
- Mobile responsive design
- Working contact page
- Blog with pagination (Pro version)
- View transitions (Pro version)
- Features page, integrations page (Pro)
- Elegant 404 page (Pro)
- Vercel + Netlify one-click deploy

## Stack
- Astro, Tailwind CSS
- TypeScript, MDX
- Prettier
- Vercel, Netlify ready
