# astrowind

> A free template using Astro 5 and Tailwind CSS. Astro starter theme.

## What's Inside
- Astro 5.0 + Tailwind CSS
- Production-ready PageSpeed scores
- Dark mode and RTL support
- Fast SEO-friendly blog with RSS
- MDX support, categories & tags
- Image optimization with Astro Assets + Unpic
- Automatic sitemap generation
- Open Graph tags
- Built-in analytics (Google Analytics, Splitbee)
- Docker support

## Stack
- Astro 5, Tailwind CSS
- TypeScript, MDX
- ESLint, Prettier
- Netlify, Vercel ready
