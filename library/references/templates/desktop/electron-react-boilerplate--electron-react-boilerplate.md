# Electron React Boilerplate

> A Foundation for Scalable Cross-Platform Apps

## What's Inside
- Electron with React and React Router
- Webpack with React Fast Refresh
- TypeScript support
- Hot module replacement
- Cross-platform packaging for production
- ESLint and Prettier configuration
- GitHub Actions CI/CD
- OpenCollective for sponsorship
- Discord community support
- Comprehensive documentation site

## Stack
- TypeScript, JavaScript, CSS, EJS
- Electron, React, React Router
- Webpack, React Refresh
- ESLint, Prettier
- npm for package management
