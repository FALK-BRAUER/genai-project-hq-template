# tauri-ui

> Tauri + shadcn/ui starter. Pick your framework and start building — Vite, Next.js, React Router, Astro, TanStack Start

## What's Inside
- Tauri v2 with shadcn/ui integration
- Multiple framework adapters (Vite, Next.js, React Router, Astro, TanStack Start)
- Desktop-ready defaults (no startup flash, external links in browser, no overscroll)
- Built-in debug panel for inspecting app state (dev-only)
- Native window controls with tauri-controls
- GitHub Actions release workflow
- Smaller binary output (~65% reduction)
- Optional starter dashboard template
- Rust invoke examples
- Dockable debug panel with layout persistence

## Stack
- TypeScript, JavaScript, Go Template, MDX, CSS
- Tauri v2, Rust
- shadcn/ui, Radix UI, Tailwind CSS
- Vite, Next.js, React Router, Astro, TanStack Start
- Bun for package management
- Turborepo for monorepo
