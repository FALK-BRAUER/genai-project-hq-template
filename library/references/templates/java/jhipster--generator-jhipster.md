# jhipster/generator-jhipster

> JHipster is a development platform to quickly generate, develop, & deploy modern web applications & microservice architectures

## What's Inside
- Yeoman generator for full-stack applications
- Spring Boot backend generation
- Frontend: Angular, React, Vue support
- Microservices architecture generation
- Docker + Kubernetes deployment configs
- CI/CD pipelines (GitHub Actions, GitLab, Jenkins)
- Database schemas (SQL + NoSQL)
- Authentication: JWT, OAuth2, Session
- SonarQube integration
- Blueprint system for customization
- Entity generation with relationships

## Stack
- Node.js, TypeScript
- Java 21/25, Spring Boot 3.x
- Angular, React, or Vue
- Maven, Gradle
- Docker, Kubernetes
- SQL (PostgreSQL, MySQL) and NoSQL (MongoDB, Cassandra)
- Liquibase, Flyway
