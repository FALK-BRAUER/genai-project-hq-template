# CleanArchitecture

> Clean Architecture Solution Template: A proven Clean Architecture Template for ASP.NET Core 10 with full and minimal options.

## What's Inside
- Two template options: Full (multi-project) and Minimal (single-project vertical slices)
- Domain-Driven Design patterns (entities, aggregates, value objects, domain events)
- FastEndpoints for API endpoint organization
- Repository pattern with specifications
- .NET Aspire support for cloud-ready development
- Comprehensive test projects (unit, functional, integration)

## Stack
- C# (100%)
- .NET 10 / ASP.NET Core
- Entity Framework Core
- FastEndpoints library
