# vue-element-admin
> A magical vue admin. An out-of-box UI solution for enterprise applications.

## What's Inside
- Login/logout with two-step authentication
- Permission authentication (page, directive, config level)
- Dynamic routing and sidebar management
- i18n internationalization support
- Rich text, Markdown, and JSON editors
- Excel export/upload/visualization
- Dynamic tables with drag-and-drop and inline edit
- Dashboard with ECharts integration
- Error pages (401, 404)
- Tags-view and breadcrumb navigation

## Stack
- Vue.js 2.x
- Element UI
- Vuex for state management
- Vue Router for navigation
- Axios for HTTP requests
- ESLint, Babel, PostCSS
