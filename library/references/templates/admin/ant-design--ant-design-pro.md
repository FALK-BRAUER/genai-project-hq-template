# ant-design-pro
> An out-of-box UI solution for enterprise applications as a React boilerplate.

## What's Inside
- TypeScript support for application-scale JavaScript
- Block templates for page building
- Neat design following Ant Design specification
- Common templates for enterprise applications
- Responsive design for variable screen sizes
- Customizable theming with simple config
- Built-in i18n solution
- Best practices for healthy code workflow
- Mock development solution
- UI tests (unit and e2e)
- Umi 4 and Umi 3 framework support

## Stack
- React
- Ant Design
- Umi framework
- Dva for state management
- TypeScript
- Biome for linting
- GitHub Actions for CI
