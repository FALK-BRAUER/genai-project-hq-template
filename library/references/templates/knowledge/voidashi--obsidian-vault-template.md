# voidashi/obsidian-vault-template

> Simple and efficient template to build your second brain in Obsidian

## What's Inside
- Controlled two-level hierarchy (max 2 clicks to any note)
- Dashboard as central hub
- Personal, Knowledge, Projects, References structure
- Properties-based organization with YAML frontmatter
- Comprehensive tagging system (type, context, theme, status)
- Maps of Content (MOCs) for theme organization
- Ready-to-use note templates
- Zettelkasten, PARA, GTD methodology integration
- No required plugins (works with core Obsidian)

## Stack
- Obsidian (markdown vault)
- YAML frontmatter
- Markdown with wikilinks
