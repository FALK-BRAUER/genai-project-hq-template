# Chrome Extension Boilerplate React Vite

> Chrome Extension Boilerplate with React + Vite + TypeScript

## What's Inside
- React with TypeScript for extension development
- Vite with Rollup for fast builds
- Turborepo for monorepo management
- Tailwind CSS for styling
- Chrome Manifest V3 support
- Custom i18n package for internationalization
- Custom HMR plugin for development
- End-to-end testing with WebdriverIO
- Multiple page types (popup, options, content scripts, devtools, side-panel)

## Stack
- TypeScript, React, Vite
- Turborepo, Tailwind CSS
- ESLint, Prettier
- WebdriverIO for e2e testing
- pnpm for package management
