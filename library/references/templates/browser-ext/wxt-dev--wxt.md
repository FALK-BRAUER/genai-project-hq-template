# WXT

> Next-gen Web Extension Framework - It's like Nuxt, but for Web Extensions

## What's Inside
- Support for all browsers (Chrome, Firefox, Safari, Edge)
- Both MV2 and MV3 manifest support
- Dev mode with HMR and fast reload
- File-based entrypoints
- TypeScript with auto-imports
- Frontend framework agnostic (Vue, React, Svelte)
- Module system for code reuse
- Bundle analysis
- Automated publishing workflow
- Template scaffolding CLI (npx wxt@latest init)

## Stack
- TypeScript
- Vite, Rollup
- Multiple frontend frameworks supported
- pnpm workspace
- Vitest for testing
