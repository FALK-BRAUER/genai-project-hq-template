# create-t3-turbo

> Clean and simple starter repo using the T3 Stack along with Expo React Native

## What's Inside
- Turborepo monorepo setup
- Expo app with SDK 54, React Native 0.81, React 19
- Next.js 15 app with React 19
- Tanstack Start app (optional)
- tRPC v11 for typesafe APIs
- Better Auth for authentication
- Drizzle ORM with Supabase/PostgreSQL
- shadcn-ui components
- Tailwind CSS v4
- Shared tooling (eslint, prettier, typescript configs)
- EAS Build + Submit for Expo deployment
- Auth proxy for OAuth in development

## Stack
- TypeScript 89%, CSS 8%, JavaScript 2%
