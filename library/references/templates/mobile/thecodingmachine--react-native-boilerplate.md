# react-native-boilerplate

> A React Native template for building solid applications 🐙, using JavaScript 💛 or Typescript 💙 (you choose).

## What's Inside
- Optimized architecture with separation of concerns
- UI and business logic decoupling
- Full documentation for all code
- JavaScript or TypeScript support
- Tanstack Query for data fetching
- Zod for validation
- Modern React Native CLI setup
- Automated CI checks
- Post-init configuration script
- Component structure with clear patterns

## Stack
- TypeScript 70%, JavaScript 22%, Swift 3%, Kotlin 3%
- React Native, React
- Tanstack Query, Zod
- Yarn (v2+)
