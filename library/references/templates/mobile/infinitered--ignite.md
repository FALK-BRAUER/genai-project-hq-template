# ignite

> Infinite Red's battle-tested React Native project boilerplate, along with a CLI, component/model generators, and more! 9 years of continuous development and counting.

## What's Inside
- React Native 0.81 with React 19
- Expo SDK 55 (optional modules)
- React Navigation v7
- TypeScript 5 with static typechecking
- RN Reanimated v4 for animations
- MMKV v3 for state persistence
- apisauce v3 REST client
- Jest v29 for testing
- Maestro for E2E UI testing
- Reactotron RN for debugging
- i18n support with RTL
- Component library with theming
- CLI with component/model generators

## Stack
- TypeScript 97%, JavaScript 3%
- React Native, Expo, React
- React Navigation, Reanimated
- MMKV, apisauce, Jest
- Hermes JS engine
