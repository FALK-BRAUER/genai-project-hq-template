# clean-architecture-swiftui

> SwiftUI sample app using Clean Architecture. Examples of working with SwiftData persistence, networking, dependency injection, unit testing, and more.

## What's Inside
- Clean Architecture with decoupled layers
- Presentation layer (SwiftUI views)
- Business Logic layer (Interactors)
- Data Access layer (Repositories)
- Redux-like centralized AppState
- Native SwiftUI dependency injection
- Programmatic navigation with deep links
- SwiftData for data persistence
- async/await networking layer
- Full test coverage including UI (ViewInspector)
- System event handling (didBecomeActive, etc.)

## Stack
- Swift 100%
- SwiftUI, Combine
- SwiftData
- ViewInspector for UI testing
