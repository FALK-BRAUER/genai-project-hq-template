# Bullet Train

> The Open Source Ruby on Rails "super scaffolding" starter app for building SaaS applications quickly.

## What's Inside
- Multi-tenancy with team-based architecture
- User authentication and authorization
- Subscription billing integration (Stripe)
- RESTful API scaffolding
- Super scaffolding generators for rapid CRUD creation
- Admin panel scaffolding
- Webhook handling infrastructure
- Audit logging
- Role-based access control
- Internationalization (i18n) ready
- Email notification system
- File upload handling (Active Storage)

## Stack
- Ruby 3.3+
- Rails 7.x/8.x
- PostgreSQL
- Redis
- Sidekiq (background jobs)
- Tailwind CSS
- Hotwire (Turbo + Stimulus)
- Stripe (payments)
- RSpec (testing)
