# dawn

> Shopify's first open source reference theme with Online Store 2.0 features and performance built-in.

## What's Inside
- HTML-first, JavaScript-only-as-needed approach
- Online Store 2.0 features (app blocks, metafields)
- Performance-optimized with Lighthouse CI
- Progressive enhancement for older browsers
- Server-rendered with Liquid templating
- Theme Check integration for linting

## Stack
- Liquid (63.3%), CSS (24.0%), JavaScript (12.7%)
- Shopify Theme framework
- Theme Check for validation
