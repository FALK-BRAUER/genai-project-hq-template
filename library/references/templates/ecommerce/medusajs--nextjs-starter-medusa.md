# nextjs-starter-medusa

> A performant frontend ecommerce starter template with Next.js 15 and Medusa V2 backend.

## What's Inside
- Next.js 15 with App Router and Server Components
- Full ecommerce features (products, cart, checkout, user accounts)
- Stripe payment integration
- Tailwind CSS styling
- TypeScript with full type safety
- Server Actions for mutations

## Stack
- TypeScript (97.7%), JavaScript, CSS
- Next.js 15
- Tailwind CSS
- Medusa V2 backend
