# vitepress

> Vite & Vue powered static site generator — spiritual successor to VuePress built on top of Vite.

## What's Inside
- Vue 3-powered SSG with Vite for fast dev server and HMR
- Markdown-based content with Vue component interpolation
- Built-in theme with responsive layout
- Full-text search integration
- Automatic code syntax highlighting
- Playgrounds with instant preview via StackBlitz integration

## Stack
- TypeScript (56.5%), Vue (30.3%), CSS (12.0%)
- Vite build tool
- Vue 3 framework
