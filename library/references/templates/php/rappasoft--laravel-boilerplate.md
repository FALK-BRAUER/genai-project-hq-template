# Laravel Boilerplate

> A massive head start on any size web application with backend admin panel, user management, and extensive features out of the box.

## What's Inside
- Backend admin panel built on CoreUI
- Spatie Permission package for role/permission authorization
- Frontend scaffold on Bootstrap 4
- Two-factor authentication
- User impersonation (admin can act as any user)
- Searchable/sortable tables via Laravel Livewire Tables
- Multi-lingual support (20+ languages)
- Timezone support per user
- Demo mode for showcasing features
- Activity logging
- Social login integrations

## Stack
- PHP 8.x
- Laravel 8.x/9.x
- Bootstrap 4
- Livewire
- Alpine.js
- MySQL/PostgreSQL
- Redis
