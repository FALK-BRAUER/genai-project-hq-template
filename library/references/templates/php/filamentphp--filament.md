# filament

> A powerful open-source UI framework for Laravel • Build and ship apps & admin panels fast with Livewire

## What's Inside
- Tables with filtering, actions, bulk operations
- Forms with reusable, state-aware components
- Infolists for read-only record views
- Notifications system
- Dashboard widgets
- Action modals
- Complete admin panel framework
- Extensive documentation

## Stack
- PHP 87%, Blade 5%, JavaScript 5%
- Laravel 11+, Livewire 3
- Tailwind CSS, Alpine.js
- TALL Stack (Tailwind, Alpine, Laravel, Livewire)
