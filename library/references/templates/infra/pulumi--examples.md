# pulumi/examples

> Infrastructure, containers, and serverless apps to AWS, Azure, GCP, and Kubernetes... all deployed with Pulumi

## What's Inside
- 200+ infrastructure examples across AWS, Azure, GCP, Kubernetes
- Multi-language support: TypeScript, Python, Go, C#, F#, Java
- Serverless architectures (Lambda, Azure Functions, Cloud Functions)
- Container deployments (ECS, Fargate, AKS, GKE)
- Database patterns (RDS, CosmosDB, Cloud SQL)
- Networking examples (VPC, VNet, load balancers)
- AI/ML infrastructure (LangServe deployments)

## Stack
- Languages: TypeScript, Python, Go, C#, F#, Java
- Framework: Pulumi Infrastructure as Code
- Clouds: AWS, Azure, GCP, Kubernetes, Oracle Cloud, DigitalOcean
