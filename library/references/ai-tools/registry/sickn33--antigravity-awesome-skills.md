# antigravity-awesome-skills

> Installable GitHub library of 1,400+ agentic skills for Claude Code, Cursor, Codex CLI, Gemini CLI, Antigravity, and more — battle-tested, high-performance skills including official skills from Anthropic and Vercel.

## What's Inside
- 1,400+ agentic skills covering all major domains: architecture, development, testing, security, data, AI/ML, DevOps, design, writing, research, and more
- Official skills from Anthropic and Vercel mirrored and curated
- 1272 skills in catalog with tags and triggers for automatic activation
- Skill bundles for common workflows (vibeship-spawner, idea-darwin, faf-skills)
- Community contributions merged via PR system (Satori skill pack, idea-darwin, faf-skills)
- Plugin mirrors for multiple AI coding tools
- Compatibility data for Claude Code, Cursor, Codex CLI, Gemini CLI, Antigravity
- Generated catalog and skill index for discovery
- Documentation for bundles, workflows, and tool-specific best skills

## Stack
- Python
- Markdown (SKILL.md files)
- YAML frontmatter for skill metadata
- Shell scripts for installation
