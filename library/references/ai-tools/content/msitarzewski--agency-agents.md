# agency-agents

> A complete AI agency at your fingertips — 144 specialized AI agent personalities across 12 divisions, from frontend wizards to Reddit community ninjas, each with unique voice, workflows, and deliverables.

## What's Inside
- 144 specialized agents across 12 divisions (Engineering, Design, Paid Media, Sales, Marketing, Product, Project Management, Testing, Support, Spatial Computing, Specialized, Game Development)
- Personality-driven agents with unique communication styles and approaches
- Deliverable-focused: real code, processes, measurable outcomes
- Multi-tool integration: Claude Code, Cursor, Aider, Windsurf, Gemini CLI, OpenCode, Kimi Code, GitHub Copilot
- Engineering division: frontend, backend, mobile, AI, DevOps, security, embedded, Solidity, technical writer
- Marketing division: growth hacker, TikTok strategist, Reddit community builder, China market specialists (Douyin, Xiaohongshu, Weibo, Bilibili, Zhihu)
- Specialized division: agents orchestrator, MCP builder, blockchain security auditor, compliance auditor, cultural intelligence strategist
- Game development division: Unity, Unreal Engine, cross-engine agents (designer, level designer, technical artist, audio engineer)
- Spatial computing: visionOS, XR, macOS Spatial/Metal, WebXR
- Scripts for multi-tool installation and conversion
