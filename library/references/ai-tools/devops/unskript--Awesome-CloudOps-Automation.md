# unskript/Awesome-CloudOps-Automation

> Cloud-ops automation runbooks that are ready to use. Build your own automations using the hundreds of drag and drop actions included. Built on Jupyter Notebooks, jumpstarts your SRE RunBook creation.

## What's Inside
- 61 pre-built xRunbooks for incident response, health checks, automation
- 476 Actions across 40+ integrations
- Cloud providers: AWS, Azure, GCP
- Databases: PostgreSQL, MySQL, MongoDB, Redis, MSSQL, Snowflake, ElasticSearch, OpenSearch
- DevOps tools: Kubernetes, Docker, Terraform, Jenkins, Airflow, Nomad, Kafka
- Monitoring: Prometheus, Grafana, Datadog, Splunk, Zabbix, Pingdom
- Communication: Slack, MS Teams, Jira, Opsgenie, ServiceNow
- Generative AI Action Creation with OpenAI/ChatGPT integration
- Jupyter-based runtime environment

## Stack
- Jupyter Notebooks (57.5%)
- Python (41.8%)
- Docker runtime
- Helm charts for deployment
- Apache-2.0 license
