# claude-crash-course-templates
> Three essential templates for rapidly building and deploying AI-driven applications using Claude

## What's Inside
- Template 1: Generate Master Plan (1_generate_master_plan.txt) - defines app structure and purpose, generates masterplan.md
- Template 2: Stub Out Project (2_stub_out_project.txt) - creates project skeleton with basic architecture
- Template 3: Fully Code Out Implementation (3_fully_code_out_implementation.txt) - transforms stubs into production-ready code
- Structured workflow for 10x faster development
- MIT License

## Stack
- Text templates (.txt files)
- Works with any stack (Next.js, Python, etc.)
