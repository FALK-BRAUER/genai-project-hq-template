# claude-code-ultimate-guide

> A tremendous feat of documentation covering Claude Code from beginner to power user, with production-ready templates for Claude Code features.

## What's Inside
- 24K+ lines across 16 specialized guides
- 226 production templates (agents, commands, hooks, skills)
- 271-question quiz (9 categories, 4 skill profiles)
- 41 Mermaid diagrams (architecture, multi-agent, security, cost optimization)
- Security threat database (24 CVEs, 655 malicious skills)
- MCP server for guide access from any Claude Code session
- Methodology guides (TDD, SDD, BDD with AI collaboration)
- Agent Teams coverage (v2.1.32+ experimental)
- 115 resource evaluations (5-point scoring)
- Role-based learning paths (Junior, Senior, Power User, PM, DevOps, Designer)
- Interactive onboarding system (4 questions → custom learning path)
- Talk pipeline for presentation preparation (6 stages from raw material to revision sheets)
- Landing page generator for open-source projects
- Design patterns detector for GoF patterns in TypeScript/JavaScript

## Stack
- Markdown
- Mermaid (diagrams)
- MCP server (Node.js/npx)
- Astro (landing site at cc.bruniaux.com)
- PDF/EPUB exports
