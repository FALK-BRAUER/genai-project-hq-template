# GenAI_Agents

> Comprehensive repository for Generative AI Agent development — from beginner conversational bots to advanced multi-agent systems with 45+ implementations.

## What's Inside
- 45+ agent implementations across 10 categories (beginner, business, creative, analysis, news, shopping, task management, QA, advanced)
- LangGraph-based workflows for stateful multi-agent coordination
- Educational agents: ATLAS (academic task system), scientific paper agent, Chiron (Feynman learning)
- Business agents: customer support, essay grading, travel planning, contract analysis, E2E testing
- Creative agents: GIF generator, TTS poem generator, music compositor, content intelligence
- Analysis agents: memory-enhanced conversational, self-improving, internet search, sales call analyzer
- Multi-agent collaboration patterns with AutoGen and CrewAI
- Specialized agents: EU Green Deal bot, murder mystery game, car buyer agent

## Stack
- Python, Jupyter Notebooks
- LangChain, LangGraph, AutoGen, CrewAI, OpenAI Swarm
- Pydantic, ChromaDB, Tavily Search
- Playwright (browser automation), OpenAI APIs (GPT-4, TTS, DALL-E)
