# agents-towards-production

> The open-source playbook for turning AI agents into real-world products — end-to-end, code-first tutorials covering every layer of production-grade GenAI agents.

## What's Inside
- Production-ready tutorials for orchestration, memory, observability, deployment, and security
- Tool integration with OAuth2 (Arcade), web data (Bright Data), real-time search (Tavily)
- Memory systems with Redis (dual-memory) and Mem0 (hybrid vector + graph)
- Deployment guides for Docker, AWS Bedrock AgentCore, RunPod GPU
- Security guardrails with LlamaFirewall and prompt injection testing
- Multi-agent coordination with A2A protocol
- Agent frameworks: LangGraph, MCP, FastAPI APIs, Kotlin/Koog
- Evaluation and tracing with LangSmith and IntellAgent
- UI development with Streamlit chat interfaces

## Stack
- Python, Jupyter Notebooks, Kotlin
- LangChain, LangGraph, FastAPI, Streamlit
- Redis, Mem0, ChromaDB, Ollama
- Docker, AWS Bedrock, RunPod
