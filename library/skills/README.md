# library/skills/

Skill libraries loaded via `/read` into Claude Code sessions. Each skill is a self-contained SKILL.md with instructions and context.

- `business-ops/` — accounting, HR, contracts, branded-docs, solo-ops
- `knowledge/` — research and knowledge management skills
- `orchestration/` — multi-agent coordination skills
- `task/` — task planning and execution skills
- `_candidates/` — skills under evaluation, not yet promoted
- `CATALOG.md` — index of all available skills
