# templates/

Project scaffolding templates used by `/new-project`. Each subdirectory is one template.

- `nextjs-app/` — Next.js App Router + TypeScript + Tailwind + shadcn/ui
- `fastapi-service/` — Python FastAPI service with Pydantic and async patterns
- `consulting-report/` — branded consulting deliverable template

Each template contains a `manifest.json` describing its structure and required inputs.
