# global-config/opencode/

MCP server configuration for opencode. Mirrors the Claude Code global MCP list.

- `mcp-servers.json` — MCP definitions in opencode format; deploy to `~/.opencode/mcp.json` or merge into your existing opencode config

## Alignment rule

Keep `mcp-servers.json` in sync with the Claude Code global MCP config. When an MCP is added or removed from Claude Code globally, update this file to match.
