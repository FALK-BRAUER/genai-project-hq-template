# global-config/openclaw/

> **Optional / advanced.** OpenClaw is a setup for running a dedicated AI agent on a separate macOS user account (e.g. a local Mac Mini acting as an always-on agent). Skip this if you're running everything under your own user account.

Configuration for the openclaw AI agent runtime.

- `openclaw.json.template` — config template (fill credentials before deploying)
- `SOUL.md` — system prompt / personality definition for the openclaw agent
- `vetted-skills/` — skills approved for use by the openclaw agent
