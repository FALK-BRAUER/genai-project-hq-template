# UI UX Pro Max

- **Type:** Skill
- **Command:** Auto-triggers on UI/UX tasks
- **Source:** [nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)
- **Scope:** Global
- **Status:** Active

## What It Does

Comprehensive design intelligence for web and mobile. Provides instant access to:
- 50+ visual styles (glassmorphism, brutalism, minimalism, etc.)
- 161 color palettes with product-type matching
- 57 font pairings with Google Fonts integration
- 99 UX guidelines (accessibility, touch, performance, animation)
- 25 chart type recommendations
- 10 technology stacks (React, Next.js, Vue, Svelte, SwiftUI, etc.)

## Triggers

Activates automatically when tasks involve:
- Designing new pages (landing, dashboard, admin, SaaS, mobile)
- Creating/refactoring UI components
- Choosing colors, typography, spacing, layouts
- Reviewing UI for UX, accessibility, or visual consistency
- Navigation, animations, responsive behavior

## Tips & Gotchas

- 658-line SKILL.md — very comprehensive, covers WCAG, Apple HIG, Material Design
- Priority system: Accessibility (P1) → Touch (P2) → Performance (P3) → Style (P4) → Layout (P5)
- Includes 2025 trends: Anti-Polish, Tactile Digital, Nature Distilled, etc.
