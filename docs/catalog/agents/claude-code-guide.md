# Claude Code Guide

- **Type:** Agent
- **Subagent Type:** `claude-code-guide`
- **Scope:** Global (built-in)
- **Status:** Active

## What It Does
Answers questions about Claude Code itself -- features, hooks, slash commands, MCP servers, settings, IDE integrations, and configuration. Acts as a built-in reference manual for the tool you're using.

## Tools Available
Glob, Grep, Read, WebFetch, WebSearch.

## When to Use
- "How do I configure hooks in Claude Code?"
- "What slash commands are available?"
- "How does MCP server configuration work?"
- Understanding Claude Code settings, IDE integration, or capabilities

## When NOT to Use
- Questions about your project's code (use explore or general-purpose)
- Tasks that require editing files
- General programming questions unrelated to Claude Code
