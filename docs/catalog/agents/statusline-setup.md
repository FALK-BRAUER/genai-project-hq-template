# Statusline Setup

- **Type:** Agent
- **Subagent Type:** `statusline-setup`
- **Scope:** Global (built-in)
- **Status:** Active

## What It Does
Configures Claude Code status line settings. Reads current configuration, applies requested changes to the status line display in the terminal.

## Tools Available
Read, Edit.

## When to Use
- Customizing the Claude Code status line display
- Enabling or disabling status line elements
- Troubleshooting status line rendering issues

## When NOT to Use
- Any task unrelated to the status line
- General Claude Code configuration (use claude-code-guide for guidance first)
