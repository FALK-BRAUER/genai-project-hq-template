# tools/scripts/

Scheduled and on-demand shell scripts that power vibe-hq automation.

Each `run-*.sh` script corresponds to a launchd job. Run manually or via cron. Logs go to `tools/logs/`.

Key scripts: `run-doctor.sh`, `run-privacy-audit.sh`, `run-cost-report.sh`, `run-sync-costs.sh`, `run-learning-harvest.sh`.
